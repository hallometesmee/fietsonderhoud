Strava koppelen — stap voor stap

Je hebt een gratis Strava API-applicatie nodig. Dit doe je één keer:

  1. Ga naar strava.com/settings/api
  2. Maak een applicatie aan (naam maakt niet uit, bijv. "Mijn Fiets Tracker")
  3. Zet bij Authorization Callback Domain: localhost
  4. Kopieer je Client ID en Client Secret hieronder
  5. Klik "Autoriseer met Strava" — log in en geef toestemming
  6. De kilometerstand wordt daarna automatisch ingeladen per fiets

Vul ook de Strava Bike ID per fiets in. Die vind je via Strava → Instellingen → Mijn uitrusting. De ID staat in de URL (bijv. b12345678).
